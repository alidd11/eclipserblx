🚀 Quick Start

🎨 Customisation Guide

Changing Your Game Name & Description

Search for the following text in the file and replace with your own:

<!-- Find this in the Hero section -->
<h1>YOUR GAME NAME</h1>
<p>Your game description goes here...</p>

Changing Colours

The template uses CSS custom properties (variables) at the top of the file. Edit these to change the entire colour scheme:

/* Find the :root section near the top */
:root {
  --primary: #00f0ff;     /* Main accent (cyan) */
  --secondary: #bf5af2;   /* Secondary accent (purple) */
  --bg-dark: #0a0a1a;     /* Background colour */
  --card-bg: #12122a;     /* Card backgrounds */
}
💡 Tip: Use a colour picker like coolors.co to generate matching colour palettes.

Updating Stats

Find the stats section and change the numbers:

<div class="stat">
  <span class="stat-number">24.5K</span>
  <span class="stat-label">Active Players</span>
</div>

Changing Features

Each feature card follows this pattern — edit the icon, title, and description:

<div class="feature-card">
  <div class="feature-icon">🏎️</div>
  <h3>Your Feature Title</h3>
  <p>Description of this feature...</p>
</div>

Adding Screenshots

Replace the placeholder gallery images with your own Roblox screenshots:

<!-- Replace the src with your image URLs -->
<img src="your-screenshot.jpg" alt="Screenshot">
💡 Tip: Use Roblox Studio's screenshot feature (F12) to capture high-quality in-game images. Recommended size: 1920×1080.

Updating Social Links

Find the Community section and replace the links:

<!-- Update these URLs -->
<a href="https://discord.gg/YOUR-SERVER">Join Discord</a>
<a href="https://www.roblox.com/groups/YOUR-GROUP">Roblox Group</a>
<a href="https://x.com/YOUR-HANDLE">Follow on X</a>

Updating the Changelog

Add new entries to the changelog section following this format:

<div class="changelog-entry">
  <span class="version-tag">v1.1</span>
  <span class="update-title">Your Update Title</span>
  <span class="update-tag new">New</span>
</div>

📁 File Structure

The template is a single self-contained HTML file. All CSS and JavaScript is embedded inline — no external files, frameworks, or build tools are needed.

⚠️ Important: Do not remove the <meta charset="UTF-8"> or <meta name="viewport"> tags — these ensure proper display on all devices.

🔧 Advanced Customisation

Adding Google Analytics

Paste your Google Analytics tag just before </head>:

<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXX"></script>

Adding a Custom Favicon

Add this line inside the <head> section:

<link rel="icon" href="favicon.ico" type="image/x-icon">

Connecting a Custom Domain

After uploading to your host, follow their documentation to connect a custom domain. Most hosts provide free SSL certificates automatically.
